What is OSI model:

OSI model, stands for Open Systems Interconnection model, is a conceptual framework created by the International Organization of Standardization. It is designed to characterize communication functions of diverse computing systems by dividing network communication into 7 abstract layers: physical layer, datalink layer, network layer, transport layer, session layer, presentation layer, and application layer.

Data Link Layer

When you think of Data Link layer, consider it as the manager who control data to be transfer from point to point. However, it is only in charge of the data that transfer under the same network. What is interesting is that Data Link layer takes the packege from the network layer and break them down into frames. Let's now talk about the major functions of Data Link layer.

    Framing:
    As I've mentioned, Data Link Layer will break the stream of bits, received from Network Layer, into a managable unit called frame.
    Physical Addressing:
    Data Link layer will add a header in front of the frame, making it recognizable if it is distributed to different systems on the network.
    Flow Control:
    What if all the cars on a highway drive with a different speed? Probably there is going to have a serious traffic jam. Just like how network stuck if a fast transmitter connets with a slow receiver. This is also why Data Link layer is important, since it prevents this problem.
    Error Control:
    To prevent from losing frames and unintentional duplications, Data Link layer add a trailer at the end of the frame to detect if anything is going wrong.
    Multiple Access:
    In order to prevent collision, Data-link layer provides mechanism such as CSMA/CD to make it possible to access a shared media from different systems.
